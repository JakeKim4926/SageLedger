# SageLedger — 모임 회비 장부 자동화

매달 모임 회비 장부 정리, 손으로 하면 번거롭죠.
**SageLedger는 카카오뱅크 모임통장 거래내역만 넣으면, 모임 장부를 알아서 정리해 주는 프로그램입니다.**

## 무엇을 해주나요

- **회원별 입금 자동 기록** — 누가 언제 얼마 냈는지 거래내역에서 찾아 장부에 채워 줍니다.
- **몰아서 낸 회비 자동 나누기** — 한 번에 여러 달치를 내도 월 회비 단위로 나눠 기록합니다 (예: 22만 원 → 2만 원씩 11칸).
- **수입·지출·이자 정리** — 입출금 내역도 함께 채워 줍니다.
- **중복 걱정 없음** — 이미 적힌 항목은 다시 적지 않아서, 여러 번 돌려도 장부가 꼬이지 않습니다.
- **잔액 맞춤 확인** — 카카오뱅크 잔액과 장부 잔액이 일치하는지 자동으로 검사합니다.
- **애매한 항목은 따로 모음** — 사람이 한 번 봐야 할 입금은 `검토결과` 시트로 빼 줍니다.

## 어떻게 쓰나요

1. 카카오뱅크 거래내역과 모임 장부 파일을 정해진 폴더에 넣습니다.
2. 프로그램을 실행합니다.
3. 정리된 장부가 결과 폴더에 새로 생깁니다. → 모임 단톡방에 올리면 끝!

> 원본 장부는 그대로 두고 **새 파일**로 만들어 주기 때문에 안전합니다.

## 지원하는 모임

현재 **FIT**, **SCM** 모임을 지원하며, 모임이 늘어나도 이름만 추가하면 똑같이 처리할 수 있습니다.

## 주의

장부에는 회원 이름·금액 같은 **개인정보**가 들어 있습니다. 결과 파일은 모임 구성원끼리만 공유해 주세요.
